# Manufacturing Productivity & Predictive Modeling 🏭📊

## Project Overview
This project analyzes garment manufacturing data to identify key factors influencing worker productivity. It employs inferential statistics to validate operational changes and regression modeling to predict future performance.

## Business Challenge
* **Hypothesis Testing:** Evaluate if new machinery effectively reduces worker overtime and operational costs.
* **Productivity Drivers:** Identify which variables (incentives, team size, SMV) have the strongest impact on factory output.
* **Temporal Analysis:** Determine if productivity fluctuates significantly across different days of the workweek.

## Technical Stack
* **Language:** Python
* **Libraries:** Pandas, Statsmodels, Scikit-learn, Matplotlib, Seaborn.
* **Statistical Methods:** Multiple Linear Regression, One-sample T-tests, P-value Analysis, Regression Diagnostics.

## Key Insights & Results
* **Regression Analysis:** Identified that **Targeted Productivity** and **Incentives** are the strongest positive predictors of actual output.
* **Operational Bottlenecks:** Found that high **SMV (Standard Minute Value)** and **Overtime** negatively correlate with efficiency, suggesting a need for task optimization.
* **Hypothesis Outcome:** Statistical testing (T-test) showed no significant reduction in overtime from new machinery in the current sample, providing a data-backed basis for strategic investment review.

## Project Structure
* `CA2.ipynb`: Jupyter Notebook containing data cleaning, visualization, and statistical modeling.
* `garments_worker_productivity.csv`: The primary dataset.
* `Final_Report.pdf`: Comprehensive analysis and management recommendations.
